### 分析指標計算

```typescript
// 互動率 = (點贊 + 評論×2 + 分享×3) / 瀏覽數 × 100%
engagement_rate = (likes + comments*2 + shares*3) / views * 100

// 熱度評分 = 互動率(40%) + 情感(20%) + 關鍵詞(40%)
trend_score = engagement_rate*0.4 + sentiment_score*0.2 + keywords_score*0.4

// 變現潛力評估
if (trend_score > 70 && engagement_rate > 5%) → "high"
else if (trend_score > 50 && engagement_rate > 3%) → "medium"
else → "low"
```

## 🚀 快速開始

### 本地開發

```bash
# 1. 克隆項目
git clone https://github.com/Demianyuen/xhs-downloader-web.git
cd xhs-downloader-web

# 2. 安裝依賴
npm install

# 3. 啟動開發服務器
npm run dev

# 4. 訪問應用
# 首頁: http://localhost:3000
# 分析: http://localhost:3000/analyze
# 歷史: http://localhost:3000/history
```
